Git

发表于 2019-10-24 | 分类于 随笔

在github上创建仓库

进入项目文件夹(安装git的前提下)

git Bash Here

git init

git add . (add与.有空格)

git commit -m” description”

得到你的SSH 链接

git remote add origin SSH 链接

如果仓库不是空的

git pull –rebase origin master

git push origin master

如果仓库是空的(先前没有提交过并且也没有markdown文件)

git push -u origin master

以后更新只要

git push origin master

其他常用操作

git status

git log

git branch -a
